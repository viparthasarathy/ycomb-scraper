I started using Impala this summer on AWS Elastic Mapreduce. Previously I'd been using Hive. We had at least a 100x speedup in most of the work we do because of it. Things that would have been impossible before (near realtime classification) are now feasible with mini-batches. My only (major) gripe is that AWS only supports several versions back. This makes it difficult to troubleshoot problems and find documentation. That said, the benefits far far outweight the cost.
